## Title

Maitri, a safety verified Gemma 4 maternal referral co pilot

## Description

Maitri is a seven agent maternal triage system built on Gemma 4. An independent verifier rejects unsupported medical claims before they ever reach the community health worker, and an append only audit log records every model call, every cited evidence chunk, and every tool argument.

In this three minute demo you will see the hero loop. The Specialist agent drafts a risk assessment. The Verifier rejects an unsupported drug dose claim. The Optimizer writes a rewrite hint. The Specialist runs again. The second draft passes. Only then does the Referral and Facility agent pick a safe destination, and the Formatter produces a clinical card for the worker, a warm message for the mother, and a separate one for the household decision maker.

Built for the Gemma 4 Good Hackathon. Apache 2.0 license on code.
